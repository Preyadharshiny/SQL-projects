E-commerce Database Analysis
-	Designed a Relational database with Customers ,Orders, Order_details and Products table to simulate an e-commerce system.
-	Performed data analysis using SQL Joins, Subqueries and Window functions. 
-	Implemented Ranking and Running total logic using Rank() and sum() OVER for revenue analysis
-	Identified key insights such as top customers, best-selling products and and category-wise sales performance
-	Used OFFSET-FETCH for pagination and advanced filtering
